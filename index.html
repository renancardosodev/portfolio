Meus Projetos

Escalador de Jogadores

HTML | CSS | JAVASCRIPT

Este é um projeto de Gerenciador de Equipes desenvolvido com o objetivo de oferecer uma interface amigável para os usuários adicionarem ou removerem jogadores de uma equipe. Os usuários têm a capacidade de adicionar um jogador à equipe, fornecendo seu nome, posição (por exemplo, atacante, defensor, goleiro) e número da camisa, com uma mensagem de confirmação antes de adicionar o jogador. Além disso, podem remover um jogador da equipe inserindo o número da camisa, novamente com uma mensagem de confirmação.

Cadastro de Desenvolvedores

HTML | CSS | JAVASCRIPT

Este é um projeto de formulário de registro para desenvolvedores, concebido para facilitar o processo de registro de habilidades e experiências. Com uma interface simples e amigável, os usuários podem inserir seu nome completo e adicionar as tecnologias que dominam, indicando seu tempo de experiência em cada uma. O formulário foi projetado com validação de dados para garantir a precisão das informações fornecidas e é responsivo, adaptando-se facilmente a diferentes tamanhos de tela para uma experiência consistente em vários dispositivos.

Jogo da Velha

HTML | CSS | JAVASCRIPT

Este é um projeto de jogo da velha desenvolvido com o objetivo de criar uma interface intuitiva e responsiva para coletar informações de registro de jogadores. O jogo permite que os jogadores preencham seus nomes nos campos de registro e escolham entre 'X' ou 'O' para iniciar a partida. A jogabilidade continua até que um jogador vença ou haja um empate, com pontos acumulados para cada jogador com base nas vitórias. O projeto também inclui uma opção para reiniciar o jogo e visualização do placar acumulado de cada jogador para acompanhar o histórico de vitórias.
